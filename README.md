# Banket — банкетные залы Петрозаводск

Flask-приложение с лендингом по дизайну из папки `Design`. Реализация в чистом HTML и CSS (без Tailwind/React).

## Установка и запуск

```bash
# Виртуальное окружение (рекомендуется)
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate  # Linux/macOS

# Зависимости
pip install -r requirements.txt

# Запуск
python run.py
```

Откройте в браузере: **http://127.0.0.1:5000**

## Что есть на лендинге

- **Шапка**: логотип, навигация (Услуги, Залы, Меню, Бронирование, Контакты), телефон.
- **Hero**: фон-картинка, заголовок, кнопки «Забронировать зал» и «Посмотреть залы».
- **Услуги**: 4 карточки + блок «Ресторанная кухня / Сервировка / Обслуживание».
- **Залы**: 5 карточек залов с галереей фото и списком преимуществ.
- **Меню**: заголовок, ссылка на полное меню (VK), 3 типа меню (банкет, фуршет, детское).
- **Бронирование**: форма (имя, телефон, гости, дата, зал, комментарий). Отправка через Telegram Share.
- **Контакты**: телефоны, адрес/локация.
- **Подвал** и плавающая кнопка «Позвонить».

Картинки залов и меню подгружаются с Unsplash (внешние URL). Логотип — локальный `static/images/logo.svg` (можно заменить на свой).
